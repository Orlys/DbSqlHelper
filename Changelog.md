#### Version 0.9.6
- [X] Add SqlFormat , e.g:
```C#
var sql = Db.SqlFormat("select * from {1}orders{2} where id = {0}id");
//if db is sqlserver
Assert.Equal("select * from [orders] where id = @id", sql);

//if db is oracle
Assert.Equal("select * from \"orders\" where id = :id", sql); 
```
